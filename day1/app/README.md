# Bollywood Meme Flask App

## Run locally

```bash
pip install -r requirements.txt
python app.py
```

Open `http://127.0.0.1:5000/`

## What it does

- Shows the latest image posts from `r/BollywoodMemes`
- Shows Redgifs links using live search when the Redgifs API is available
- Falls back to direct Redgifs search pages if live API calls are unavailable
